Connexion à l'API

Requête :
POST /mf_connexion
Corp :
{ "mf_login": "...", "mf_pwd": "..." }
Réponse :
{ "data": { "mf_token": "...", "id": "..." }, "error": { "number": ..., "label": ... }, "execution_time": ... }
mf_token se place dans les URL pour les requêtes nécessitants une authentification (?mf_token=...)
